Posted : 2005-05-22 00:00 / Category : [開発日誌]
Google Yahoo! などのAPIは、
簡単に言うと検索結果を XML で取得するためのものだ。
で普通の検索結果は HTML で提供される。そういう戦略。
でも、msnsearch は XHTML らしい。ほら。

結果のRSSも取れるし、MSNSearch って実は使えるんじゃないか?
